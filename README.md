# PulseStadium | Intelligent Orchestrator

PulseStadium is an elite, production-hardened digital twin platform for stadium crowd management and safety orchestration.

## 🚀 Quick Start
- **Architecture**: Next.js 15 + FastAPI + Three.js.
- **Security**: Edge-based Zero-Trust headers (Middleware enforced).
- **Accessibility**: WCAG 2.2 AAA (Color-independent indicators & Synchronized Live Regions).
- **Performance**: ES2022 optimized, Gunicorn preloading, and swap-font loading.

## 📖 Documentation
For detailed architecture, configuration, and production hardening details, see the **[User Guide](USER_GUIDE.md)**.

## 🛠️ Components
- `/client`: Next.js 15 Standalone Frontend.
- `/server`: FastAPI Backend Intelligence Engine.
- `/core`: Shared Domain Logic & Types.
